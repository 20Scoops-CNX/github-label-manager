First of all, this website works by making cross-site requests to Github's api api.github.com and there is no other call is made to anywhere else.

Disable same-origin-policy

You need to disable same-origin-policy restriction of your browser. Note that this is a temporary action and you can take it back once you are done with this website.

=== How to disable same-origin-policy ===
Chrome:
open terminal/console and execute this:
chromium-browser --disable-web-security
Well if you are on Windows you may need to use chrome.exe instead
Firefox:
Type this to url bar: about:config
Then find security.fileuri.strict_origin_policy and make it false

Is my password safe?
This website authenticates to Github api via HTTP Basic Authentication but all api calls are done over SSL so your password is safe. More on Wikipedia.
Note that your password is only sent to Github and and sent encrypted.

What this thing does anyway?
You can edit, add and delete labels for your Github repos. Yes, not really much different from Github itself. But the thing is you can do those as bunches here. The main thing is that you can also copy another repos's labels into yours with or without some modifications.
